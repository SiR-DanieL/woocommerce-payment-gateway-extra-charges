A WooCommerce Extension that allow to add extra charges to your payment gateways

== Description ==

This extension will add an extra field in each WooCommerce Payment Gateway that will allow you to add extra charges on
each of your gateways. It's useful for COD payment gateway, in example, since WooCommerce does not allow this by default.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `woocommerce-payment-gateway-extra-charges` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
